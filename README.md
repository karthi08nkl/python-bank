# Bank Management System

A beginner-friendly **Bank Management System** built in Python that simulates basic banking operations through a command-line interface. The application allows users to create and manage bank accounts, perform transactions, and store account data using text files for persistent storage.

## Features

* Create a new bank account
* Secure login using account number and PIN
* Check account balance
* Deposit money
* Withdraw money
* Transfer money between accounts
* Change account PIN
* Delete an account
* View transaction history with date and time
* Persistent data storage using text files
* Unique account number generation

## Technologies Used

* Python 3

* Functions
* Variables and Data Types
* Dictionaries
* Loops (`while`)
* Conditional Statements (`if`, `elif`, `else`)
* Pattern Matching (`match-case`)
* File Handling (`read`, `write`, `append`)
